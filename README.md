# Everlane Landing Page

This project is a responsive **landing page clone** of the Everlane fashion website.
The goal of this project is to practice **frontend development skills** such as layout building, responsive design, and UI recreation.

## 🚀 Features

* Responsive navigation bar
* Hero section with promotional banner
* Clean and modern UI design
* Shop Now call-to-action button
* Fashion brand landing layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
